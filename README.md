🚀 AI Career Assistant

📌 Overview

The AI Career Assistant is a simple web application built using Streamlit that helps users search for jobs, analyze resumes, and track job applications in one place. It is designed to make the job search process easier and more organized.A professional, Streamlit-based career assistance platform designed specifically for the Indian job market. This tool helps users search for jobs, analyze resumes, and track applications in one centralized dashboard.

---

✨ Features

- 🔍 Job Search
  
  - Search jobs by role and location
  - Displays sample job listings

- 📄 Resume Analysis
  
  - Upload resume (PDF/DOCX)
  - Get basic feedback and suggestions

- 📌 Application Tracker
  
  - Add job applications
  - View all applied jobs in one place

---

🛠️ Tech Stack

- Python
- Streamlit
- Pandas

---

⚙️ Installation

1. Clone the repository:

git clone <your-repo-link>
cd AI-Career-App

2. Install required libraries:

pip install -r requirements.txt

---

▶️ How to Run

Run the following command in terminal:

streamlit run app.py

Then open in browser:

http://localhost:8501

---

⚙️ API Configurations

This application requires the following APIs to function. Ensure you have your keys ready:

1. Job Data API: (e.g., Adzuna, LinkedIn Scraper, or JSearch)

2. LLM API:(e.g., Google Gemini or OpenAI) for resume analysis.

3. PDF Generator: (e.g., ReportLab) for export functionality.
